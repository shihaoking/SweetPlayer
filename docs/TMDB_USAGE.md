# TMDB备用数据源功能使用说明

## 功能概述

SweetPlayer现在支持使用TMDB作为元数据刮削的备用数据源。当豆瓣无法找到影片信息时，系统会自动尝试从TMDB获取中文元数据。

## 配置TMDB API Key

### 1. 获取TMDB API Key

1. 访问 [TMDB网站](https://www.themoviedb.org/)
2. 注册账号（免费）
3. 进入 [API设置页面](https://www.themoviedb.org/settings/api)
4. 申请API Key（选择"Developer"类型，免费）
5. 复制生成的API Key

### 2. 配置应用

在应用程序目录下找到 `appsettings.json` 文件，添加您的API Key：

```json
{
  "Tmdb": {
    "ApiKey": "your_api_key_here"
  }
}
```

**注意**：如果不配置API Key，应用仍然可以正常使用，只是不会启用TMDB备用功能。

## 工作原理

### 刮削流程

1. **优先豆瓣**：系统首先尝试从豆瓣搜索影片元数据
2. **自动Fallback**：如果豆瓣没有结果，自动尝试TMDB
3. **中文优先**：TMDB搜索使用 `language=zh-CN` 参数获取中文数据
4. **标记来源**：在数据库中记录元数据来源（Douban或TMDB）

### 数据字段

新增数据库字段：
- `TmdbId`：TMDB影片ID（当使用TMDB数据源时）
- `DataSource`：元数据来源标识（0=Douban, 1=TMDB）

### 日志示例

```
搜索豆瓣：查询="Dune", 年份=2021
豆瓣搜索返回 0 个结果
豆瓣无结果，尝试TMDB fallback：查询="Dune", 年份=2021
TMDB搜索返回 5 个结果
选择TMDB最佳匹配：沙丘 (2021) - TMDB ID=438631
✓ 刮削成功：Dune.2021.mkv → 沙丘 (2021) [数据源: TMDB]
```

## 优势

- **更高覆盖率**：豆瓣+TMDB双数据源，更少的刮削失败
- **国际影片友好**：TMDB对国外电影支持更好
- **自动回退**：无需手动干预，自动尝试备用源
- **中文支持**：TMDB也提供中文标题和简介

## 限制

- TMDB免费API每天有请求次数限制（通常足够个人使用）
- 部分冷门影片TMDB可能只有英文信息
- 速率限制：每秒最多1个请求（已自动控制）

## 疑难解答

### TMDB不工作
- 检查API Key是否正确配置
- 查看日志确认是否有错误信息
- 确认网络连接正常（能访问TMDB API）

### 获取到英文数据
- TMDB中文数据依赖社区贡献，部分影片可能没有中文翻译
- 系统会优先返回中文数据，如无则返回原始语言

### 数据库错误
- 首次运行新版本时会自动应用数据库迁移
- 如遇到"no such column"错误，请确保应用已重启以应用迁移

## 技术细节

- **TMDB API版本**：v3
- **请求语言**：zh-CN（中文）
- **海报尺寸**：w500
- **速率控制**：与豆瓣共享刮削队列（2-5秒延迟）
- **重试机制**：429错误时等待5秒后重试一次
