## 新增需求

### 需求：自动加载本地字幕文件
系统应自动检测并加载与正在播放的视频共享相同基本文件名且位于同一目录下的字幕文件（ASS、SSA、SRT、SUB、IDX）。

#### 场景：视频在同一文件夹中有匹配的 SRT 文件
- **WHEN** 用户播放 "Movie.mkv" 且同一目录下存在 "Movie.srt"
- **THEN** 系统自动加载并显示 SRT 字幕

#### 场景：视频有多个匹配的字幕文件
- **WHEN** 用户播放 "Movie.mkv" 且同时存在 "Movie.chs.srt" 和 "Movie.cht.srt"
- **THEN** 系统加载第一个匹配的字幕，其余可供切换

#### 场景：不存在匹配的字幕文件
- **WHEN** 用户播放的视频在目录中没有匹配的字幕文件
- **THEN** 系统不带字幕播放，字幕菜单显示"无"

### 需求：手动加载本地字幕
系统应允许用户在播放期间通过文件选择器手动选择并加载本地字幕文件。

#### 场景：用户在播放期间添加本地字幕文件
- **WHEN** 用户点击"添加字幕"并通过文件选择器选择一个本地 .srt 文件
- **THEN** 系统立即加载并显示所选字幕

### 需求：通过射手网搜索在线字幕
系统应支持使用射手网 API 在线搜索字幕，可按原始视频文件名和匹配的中文电影标题进行搜索。

#### 场景：用户按文件名搜索在线字幕
- **WHEN** 用户为名为 "Interstellar.2014.2160p.mkv" 的视频点击"搜索在线字幕"
- **THEN** 系统使用文件名查询射手网 API 并显示匹配的字幕结果

#### 场景：用户按中文标题搜索在线字幕
- **WHEN** 用户为匹配到中文标题"星际穿越"的视频点击"搜索在线字幕"
- **THEN** 系统同时使用"星际穿越"查询射手网 API，并将结果与文件名搜索合并

#### 场景：用户下载并应用在线字幕
- **WHEN** 用户从在线搜索结果中选择字幕
- **THEN** 系统下载字幕文件，在本地缓存，并应用到当前播放

### 需求：字幕显示设置
系统应允许用户调整字幕显示设置，包括字体大小、字体颜色、位置和编码。

#### 场景：用户更改字幕字体大小
- **WHEN** 用户在设置中将字幕字体大小调整为"大"
- **THEN** 字幕立即以更大的字体大小渲染

### 需求：字幕轨切换
系统应允许用户在播放期间切换多个可用字幕轨（内嵌、本地和已下载的）。

#### 场景：用户从外部 SRT 切换到内嵌 ASS
- **WHEN** 用户在播放期间从字幕菜单中选择不同的字幕轨
- **THEN** 系统切换到所选字幕轨，不中断播放
