## 新增需求

### 需求：播放主流视频格式
系统应支持播放所有主流视频格式，包括 MKV、MP4、AVI、MOV、WMV、FLV、TS、M2TS、WebM 和 RMVB，并支持硬件加速解码。

#### 场景：用户播放 HEVC 编码的 MKV 文件
- **WHEN** 用户点击播放 MKV 容器中 HEVC/H.265 编码的视频文件
- **THEN** 系统使用硬件解码（D3D11VA）流畅播放视频

#### 场景：用户播放 AV1 编码文件
- **WHEN** 用户点击播放 AV1 编码的视频
- **THEN** 系统使用可用的硬件或软件解码播放视频

### 需求：沉浸式全屏播放器布局（Infuse 风格）
系统应提供无边框沉浸式播放器界面，遵循 Infuse 播放器设计。播放 UI 在独立 AppWindow（PlayerWindow）中渲染，由三个自动隐藏的覆盖区域组成：

**1. 顶部栏（信息栏）**：
- 左侧：返回按钮（返回媒体库）
- 中间：当前视频标题（电影名或"剧集名 - S01E02 集标题"）
- 右侧：设置菜单按钮（齿轮图标，打开播放器设置面板）

**2. 中央覆盖层（中央控制）**：
- 视频区域中央的大型浮动播放/暂停按钮
- 与其他控件一起在交互时出现，随之淡出

**3. 底部栏（控制栏）**：
- **进度条行**：全宽拖动条（Slider），左侧显示当前播放时间，右侧显示总时长。拖动滑块显示时间预览提示。缓冲范围在轨道上显示为较浅的区段。
- **控件行**（进度条下方）：
  - 左侧组：快退 10 秒按钮、播放/暂停按钮、快进 10 秒按钮
  - 中间组：播放速度选择器（显示当前倍速如"1.0x"）
  - 右侧组：字幕按钮（字幕选择）、音轨按钮（音轨切换）、音量滑块、全屏/窗口切换、更多菜单（三点菜单）

**自动隐藏行为**（参考 Infuse）：
- 所有覆盖层在 3 秒无鼠标/键盘操作后自动隐藏
- 控件隐藏时鼠标光标也一并隐藏
- 任何鼠标移动或键盘按键立即显示所有覆盖层
- 单击视频区域：切换控件可见性
- 双击视频区域：切换全屏

#### 场景：播放器进入全屏播放（默认窗口模式）
- **WHEN** 用户从媒体库发起视频播放，且默认播放窗口模式为 `Windowed`
- **THEN** 系统打开独立 PlayerWindow 窗口，mpv 在窗口内 GPU 零拷贝渲染视频，客户区（视频内容区域）为 1920×1130
- **THEN** 窗口显示覆盖控件，3 秒后自动隐藏

#### 场景：播放器进入全屏播放（全屏模式）
- **WHEN** 用户从媒体库发起视频播放，且默认播放窗口模式为 `FullScreen`
- **THEN** 系统打开独立 PlayerWindow 窗口，mpv 全屏 GPU 零拷贝渲染，视频填满整个屏幕
- **THEN** 显示覆盖控件，3 秒后自动隐藏

#### 场景：播放期间控件自动隐藏
- **WHEN** 播放期间 3 秒内无任何用户交互
- **THEN** 所有覆盖栏（顶部、中央、底部）和鼠标光标平滑淡出

#### 场景：用户交互显示控件
- **WHEN** 控件隐藏时用户移动鼠标、按下任意键或点击
- **THEN** 所有覆盖栏和光标平滑淡入

#### 场景：用户双击视频区域
- **WHEN** 用户双击视频区域
- **THEN** PlayerWindow 在全屏和窗口模式之间切换

#### 场景：关闭播放窗口返回媒体库
- **WHEN** 用户点击返回按钮或按 Esc 关闭 PlayerWindow
- **THEN** 系统保存当前播放进度
- **THEN** PlayerWindow 关闭，PlayerPage 桥接页自动导航返回媒体库

### 需求：播放控制（Infuse 风格）
系统应提供以下匹配 Infuse 播放器功能的播放控制：

**基础控制：**
- 播放/暂停：切换播放状态
- 快退：后退 10 秒
- 快进：前进 10 秒
- 跳转：拖动进度条到任意位置，支持帧精确跳转

**播放速度调整：**
- 支持速度：0.5x、0.75x、1.0x（默认）、1.25x、1.5x、2.0x
- 速度选择器可从底部控制栏访问
- 非 1.0x 时持续显示当前速度

**音量控制：**
- 底部控制栏的音量滑块（0-100%）
- 音频增强选项：对音量较低的视频最高可放大至 200%（参考 Infuse 的音频增强功能）
- 静音切换按钮

**画面比例/缩放：**
- 通过更多菜单：自动（默认）、填充屏幕、适应屏幕、16:9、4:3
- 支持键盘快捷键快速切换画面比例

#### 场景：用户调整播放速度
- **WHEN** 用户点击速度选择器并选择"1.5x"
- **THEN** 播放以 1.5x 速度继续，控制栏显示"1.5x"标签

#### 场景：用户使用音频增强
- **WHEN** 用户对音量较低的视频启用 150% 音频增强
- **THEN** 音频输出放大至原始音量的 150%，无削波失真

#### 场景：用户切换画面比例
- **WHEN** 用户从画面比例菜单中选择"填充屏幕"
- **THEN** 视频缩放以填满整个屏幕（必要时裁剪边缘）

### 需求：播放器设置面板（Infuse 风格标签页菜单）
系统应提供从右上角齿轮按钮访问的滑入式设置面板，按标签页组织，遵循 Infuse 菜单结构：

**标签页 1 - 视频 (Video)：**
- 画面比例选择（自动/填充/适应/16:9/4:3）
- HDR 模式指示器（适用时）

**标签页 2 - 音频 (Audio)：**
- 音轨选择（列出所有可用音轨并附带语言标签）
- 音频增强滑块（100%-200%）

**标签页 3 - 字幕 (Subtitles)：**
- 字幕轨选择（无 / 内嵌轨 / 外部加载轨）
- "添加本地字幕"按钮（打开文件选择器）
- "搜索在线字幕"按钮（触发 Shooter API 搜索）
- 字幕延迟调整（-5s 至 +5s，步长 0.1s）
- 字幕大小（小 / 中 / 大 / 超大）
- 字幕颜色（白色 / 黄色）
- 字幕位置（底部 / 顶部）

**标签页 4 - 章节 (Chapters)**（如视频包含章节）：
- 所有章节标记列表及时间戳
- 点击跳转到章节

#### 场景：用户打开设置面板
- **WHEN** 用户在播放期间点击顶部栏的齿轮图标
- **THEN** 右侧滑入面板显示标签页设置（视频/音频/字幕/章节），播放继续在后台进行

#### 场景：用户切换音轨
- **WHEN** 用户打开音频标签页并从可用轨中选择"English 5.1 DTS"
- **THEN** 音频立即切换到选定轨，不中断视频

#### 场景：用户调整字幕延迟
- **WHEN** 用户在字幕标签页将字幕延迟设置为 +0.5s
- **THEN** 所有字幕显示延迟 0.5 秒以与音频同步

#### 场景：用户跳转到章节
- **WHEN** 用户打开章节标签页并点击"Chapter 3 - The Battle (00:45:20)"
- **THEN** 播放立即跳转到 00:45:20

### 需求：剧集连播
系统应支持电视剧连续播放，遵循 Infuse 的"Up Next"设计：

- 当当前剧集播放到剩余 30 秒时，在右下角显示"Up Next"覆盖层，展示下一集信息（标题、集号）和倒计时
- 用户可点击"播放下一集"立即切换，或关闭继续观看当前集
- 当前集播放结束后，10 秒倒计时后自动播放下一集（除非用户关闭）
- 如无下一集可用，返回剧集详情页

#### 场景：剧集接近尾声
- **WHEN** 当前剧集播放至结束前 30 秒
- **THEN** 右下角出现"Up Next"卡片，显示下一集号和标题，附带"播放下一集"按钮和 10 秒倒计时

#### 场景：用户点击播放下一集
- **WHEN** 用户点击"Up Next"覆盖层中的"播放下一集"
- **THEN** 播放立即过渡到下一集，带有短暂交叉淡入

#### 场景：当前剧集结束且无用户操作
- **WHEN** 剧集播放完毕且 10 秒倒计时结束且用户未关闭
- **THEN** 系统自动播放下一集

#### 场景：无下一集可用
- **WHEN** 某季最后一集播放完毕
- **THEN** 系统显示"已播放完毕"并在 5 秒后返回剧集详情页

### 需求：播放进度记忆
系统应记住每个视频的播放位置，并提供从上次位置继续播放的选项。

#### 场景：用户重新打开之前观看过的视频
- **WHEN** 用户播放一个之前在 45 分钟处停止的视频
- **THEN** 系统显示简短提示"从 45:00 继续播放"并从该位置恢复播放（长按播放按钮可选择从头开始）

### 需求：键盘快捷键（Infuse 风格）
系统应支持全面的键盘快捷键：

| 按键 | 操作 |
|-----|------|
| Space | 播放/暂停 |
| Left Arrow | 快退 10 秒 |
| Right Arrow | 快进 10 秒 |
| Up Arrow | 音量增加 5% |
| Down Arrow | 音量减少 5% |
| F | 切换全屏 |
| Esc | 退出全屏或退出播放器 |
| M | 静音/取消静音 |
| S | 循环切换字幕轨 |
| A | 循环切换音轨 |
| [ | 降低播放速度 |
| ] | 提高播放速度 |
| Ctrl+Left | 跳转到上一章节 |
| Ctrl+Right | 跳转到下一章节 |

#### 场景：用户在播放期间按下空格键
- **WHEN** 用户在视频播放期间按下空格键
- **THEN** 播放立即暂停，中央播放按钮出现

#### 场景：用户按 S 循环切换字幕
- **WHEN** 用户在播放期间按下 S 键
- **THEN** 字幕切换到下一个可用轨（循环：无 → 轨道 1 → 轨道 2 → ... → 无）

#### 场景：用户按 ] 加速
- **WHEN** 用户在 1.0x 速度播放时按下 ] 键
- **THEN** 播放速度提升至 1.25x，短暂 OSD 显示"1.25x"

### 需求：屏幕显示 (OSD) 通知
系统应在状态变化时显示简短的屏幕通知（参考 Infuse 的 HUD 提示）：

- 音量变化：简短音量条图标 + 百分比
- 速度变化：简短"1.5x"文本
- 跳转：简短"+10s"或"-10s"指示器
- 字幕/音轨切换：简短轨名显示
- 所有 OSD 通知在 1.5 秒后自动消失

#### 场景：用户通过键盘调整音量
- **WHEN** 用户按上方向键增加音量
- **THEN** 简短 OSD 出现，显示音量图标和当前百分比（如"🔊 75%"），1.5 秒后自动消失
